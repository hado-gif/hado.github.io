Olympic Diving Score Trends

This GitHub Pages homepage introduces an interactive D3 visualization of Olympic diving competitions from 2016, 2020, and 2024. The chart compares cumulative scoring across rounds, final rankings, and dive-by-dive performance by event.

Open Visualization View Repository

What is included

  • Filters for Olympic year and diving event
  • A cumulative score progression line chart
  • Final rankings for the selected competition
  • Dive code comparison against event averages
  • Access to the underlying competition data

The visualization itself is available at vis.html.
